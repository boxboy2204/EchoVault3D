# Echo Vault 3D

A lightweight first-person 3D browser game built with plain HTML, CSS, and JavaScript.

## Run

Open [index.html](/Users/Joseph/Code/Games/First Person 3D/index.html) in a browser.

## Objective

- Click the game view to lock the mouse.
- Collect all 4 power cores.
- Avoid or shoot the roaming sentries.
- Reach the exit before the 90 second timer expires.
